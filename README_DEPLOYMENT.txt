2️⃣  FRONTEND → Vercel (3 min)
    👉 https://vercel.com/new
    - Import: AkarshYash/IP-Project
    - Root: react-frontend
    - Env: VITE_API_URL=(your Render URL)
    - Deploy → Get URL

3️⃣  UPDATE CORS (1 min)
    - Render dashboard → Environment
    - CORS_ORIGINS=(your Vercel URL)
    - Save
